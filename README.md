# wToolCore
Collection of tools I made for editing wikimedia websites.

To compile the script, just run `compile.sh` in the terminal (linux only, if on windows just do `python ./compile.py`), and copy the contents of the "compiled_script.js" file. Use "compiled_script_raw.js" for debugging more easily.


wToolCore was named this way because I thought it was a cool name. w(iki) tools, and the core bit bc... Idk that's what I got it as in my files, so uhhhh... Yea.
